Este repositorio tiene como propósito almacenar y organizar las diferentes asignaciones, prácticas y proyectos desarrollados durante la asignatura Programación I. Aquí se irán publicando los ejercicios 
realizados a lo largo del curso, los cuales abarcan los conceptos fundamentales de la programación y su aplicación práctica. El objetivo principal es llevar un registro estructurado del progreso académico 
en la materia, así como mantener disponible el código fuente de cada actividad. Con el fin de una clara revisión, mejora y referencia futura.

Dentro del repositorio se podrán encontrar implementaciones relacionadas con temas como estructuras básicas de programación, uso de funciones, control de flujo, manejo de datos y otros conceptos fundamentales.
Dichos conceptos ya vistos en la asignatura. Cada carpeta o archivo corresponderá a una práctica o proyecto específico desarrollado durante el curso.
